📌 Project Overview

This project analyzes customer behavior, churn risk, and revenue impact to help businesses improve customer retention and maximize revenue.

🎯 Business Problem

Customer churn directly impacts revenue. The goal of this project is to:

Identify high-risk customers
Quantify revenue at risk
Provide actionable retention strategies

🛠️ Tools & Technologies
SQL
Python (Pandas, NumPy)
Power BI / Tableau
Data Cleaning & Transformation

📊 Key Analysis Performed
Customer segmentation analysis
Churn rate calculation
Revenue at risk estimation
Behavioral trend analysis

🔍 Key Insights
Customers with low engagement and high support tickets show highest churn rates
A small segment of customers contributes to majority of revenue risk
Early churn indicators can be detected through usage patterns

💡 Business Recommendations
Target high-risk customers with retention campaigns
Improve onboarding experience for new users
Monitor key churn indicators proactively
