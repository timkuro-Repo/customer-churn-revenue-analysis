# Customer Churn & Revenue Analysis Dashboard

    ## Project Overview
    Analyze customer behavior, revenue, and churn risk to identify retention opportunities.

    This project is designed as a recruiter-ready data analyst portfolio project. It demonstrates how to translate raw business data into insights, KPIs, dashboards, and recommendations.

    ## Business Questions
    - Which customer segments are most likely to churn?
- What revenue is at risk from churned or high-risk customers?
- Which behavioral indicators are most associated with churn?
- What retention actions should the business prioritize?

    ## Tools & Skills
    SQL, Python, Pandas, Power BI/Tableau, Customer Segmentation, Churn Analysis

    ## Dataset
    The sample dataset is located in `/data/customer_churn.csv`.

    ## Suggested Dashboard Pages
    1. Executive Summary
    2. KPI Trends
    3. Segment / Region / Product Breakdown
    4. Root-Cause Analysis
    5. Recommendations
